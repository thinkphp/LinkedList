LinkedList Method: reverse {#LinkedList: reverse}
-------------------------------------------------

Takes an input linked list and returns a new linked list with the order of the elements reversed.

### Syntax:

    myList.reverse();

### Arguments:

- `void

### Returns:

- `void` returns a new linked list with the order of the elements reversed.

### Example: 

    mylist.insertAtHead("mootools");
    mylist.insertAtHead("jQuery");
    mylist.insertAtHead("dojo");
    mylist.insertAtHead("extjs");
    mylist.insertAtHead("yui");
    console.log(mylist.display())
    mylist.reverse();
    console.log(mylist.display())
